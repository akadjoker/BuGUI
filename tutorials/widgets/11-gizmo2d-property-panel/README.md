# 11 - Gizmo2D + Property Panel

Goal: edit a 2D transform with both direct gizmo dragging and a property panel.

Covered widgets:
- `Canvas`
- `Gizmo2D`
- `PropertyGrid`

Run:

```bash
./bin/tutorials/widgets/bugui_tutorial_widgets_11_gizmo2d_property_panel
```

Manual checks:
1. Drag translate/rotate/scale handles and confirm panel values update.
2. Edit position/rotation/scale in the panel and confirm gizmo/object update.
3. Enable snap values and verify movement becomes quantized.
