# 04 - Scroll & List Widgets

Goal: practice vertical scrolling and list selection flows.

Covered widgets:
- `ScrollView`
- `ListBox`
- `ListWidget`

Run:

```bash
./bin/tutorials/widgets/bugui_tutorial_widgets_04_scroll_list
```

Manual checks:
1. Scroll long log list and confirm smooth movement.
2. Select `ListBox` rows and verify status text changes.
3. Select `ListWidget` rows and verify selected index changes.
4. Resize window and confirm each panel remains usable.
