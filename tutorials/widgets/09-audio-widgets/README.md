# 09 - Audio Widgets

Goal: test audio-oriented widgets with animated synthetic data.

Covered widgets:
- `ADSRWidget`
- `VUMeter`
- `WaveformView`
- `SpectrumAnalyzer`
- `PianoRoll`

Run:

```bash
./bin/tutorials/widgets/bugui_tutorial_widgets_09_audio_widgets
```

Manual checks:
1. Drag ADSR handles and observe status update.
2. Verify VU and spectrum animate continuously.
3. Add/edit notes in piano roll and test playhead motion.
