# 12 - Gizmo3D + Property Panel

Goal: manipulate a 3D transform with gizmo handles and a synchronized property panel.

Covered widgets:
- `ViewportWidget`
- `Gizmo3D`
- `PropertyGrid`

Run:

```bash
./bin/tutorials/widgets/bugui_tutorial_widgets_12_gizmo3d_property_panel
```

Manual checks:
1. Switch mode (Translate/Rotate/Scale) and drag gizmo handles.
2. Edit Position/Rotation/Scale in PropertyGrid and verify cube/gizmo updates.
3. Adjust camera yaw/pitch/distance and confirm gizmo remains interactive.
