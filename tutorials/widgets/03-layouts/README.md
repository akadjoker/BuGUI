# 03 - Layout Widgets

Goal: demonstrate composition with multiple layout containers.

Covered widgets/layouts:
- `BoxLayout`
- `GridLayout`
- `FormLayout`
- `Panel`

Run:

```bash
./bin/tutorials/widgets/bugui_tutorial_widgets_03_layouts
```

Manual checks:
1. Resize window and confirm grid/form areas adapt smoothly.
2. Ensure grid buttons remain aligned in rows/columns.
3. Verify form keeps label/value structure while resizing.
