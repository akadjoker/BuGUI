# 01 - Basic Widgets

Goal: build a tiny app using base controls and signals.

Covered widgets:
- `Label`
- `Button`
- `CheckBox`
- `ProgressBar`

Run:

```bash
./bin/tutorials/widgets/bugui_tutorial_widgets_01_basic
```

Manual checks:
1. Click `Increment` and see progress/value update.
2. Disable updates with checkbox and confirm button stops changing state.
3. Click `Reset` and confirm counter returns to zero.
