# 02 - Input Widgets

Goal: connect common input controls to a live status output.

Covered widgets:
- `TextInput`
- `Slider`
- `Switch`
- `ComboBox`

Run:

```bash
./bin/tutorials/widgets/bugui_tutorial_widgets_02_inputs
```

Manual checks:
1. Type text and press Enter; submitted text must appear in status line.
2. Drag slider and verify formatted value updates.
3. Toggle switch and see `ON/OFF` reflected in output.
4. Change combobox option and confirm filter text updates.
