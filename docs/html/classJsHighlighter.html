Detailed Description

JavaScript/TypeScript syntax highlighter.

Handles: keywords, template literals (${}), single/multi-line comments, string literals, regex literals, numbers, arrow functions. Fold regions: { opens, } closes.

Compute the fold level delta for a line.

Return +1 for lines that open a fold region (e.g. { in C++), -1 for lines that close one (e.g. }), 0 otherwise. For Python, this would be based on indentation changes.

Parameters
lineIndexZero-based line number.
textThe line text.
Returns
Fold level change: +N opens N regions, -N closes N.
